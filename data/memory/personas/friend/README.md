# Friend Persona Memory

**Purpose**: Casual conversations, music, hobbies, life discussions

**Contents**:
- `conversations.jsonl` - Chat history with Friend persona
- `preferences.json` - Music preferences, favorite artists, gear
- `context.json` - Ongoing topics, references

**Lifecycle**: User-controlled (can be purged without affecting core IT knowledge)

**Privacy**: Isolated from IT Admin persona memory
